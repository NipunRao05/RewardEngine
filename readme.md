# 🎓 Event Reward DApp

A blockchain-based Event Reward Management System that rewards students with ERC-20 tokens for participating in events. Organizers can create events, distribute rewards, and students can redeem earned tokens for perks and benefits.

---

## ✨ Features

### Admin

* Manage user roles
* Configure contracts
* Pause/Unpause token system
* Burn tokens if required

### Organizer

* Create events
* Manage events
* Reward students with tokens

### Student

* Attend events
* Earn ERT tokens
* Redeem tokens for perks

---

## 🚀 Deployment

### Install Dependencies

```bash
npm install
```

### Start Hardhat Network

```bash
npx hardhat node
```

### Deploy Contracts

```bash
npx hardhat run scripts/deploy.js --network localhost
```
